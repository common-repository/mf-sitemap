== Description ==

* MF Sitemap is a simple sitemap that works perfectly.
* If URL rewrite is enabled your sitemap will be available at `/sitemap.xml` otherwise at `/?sitemap`
* Access `Plugins->MF Sitemap` to manage.
* Priorities and change frequencies are customizable through manage menu.
* Number of items per page is adjustable through manage menu.
* [POT file](http://plugins.svn.wordpress.org/mf-sitemap/trunk/languages/mf-sitemap.pot) is available for translation

== Installation ==

1. Upload the `mf-sitemap` folder to the `/wp-content/plugins/` directory
1. Activate the MF Sitemap plugin through the `Plugins` menu in WordPress
1. If URL rewrite is enabled your sitemap will be available at `/sitemap.xml` otherwise at `/?sitemap`
